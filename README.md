1. Título del proyecto:
 Trabajo Practico Integrador III

2. Descripción breve:
    Una aplicacion que permite registrar y loguearse com usuario para crear tareas y modificarlas, con estadisticas de cantidad de tareas totales, completadas y pendientes

3. Instrucciones de instalación:
    Clonar el repositorio: git clone https://github.com/Dylan-Cabrera/trabajo-practico-integrador-3-cabrera-dylan.git
    Instalar las dependencias tanto del backend como del fronend con el comando npm install
    Configurar las variables de entorno del backend en un archivo .env con los campos del .env.example
    Usar Xampp o similares para crear la base de datos
    Usar el comando npm run dev en las terminales del backend y frontend para correr el proyecto
    Abrir la url proporcianada en la consola del frontend para empezar a usar


4. Configuración del archivo .env:
    Variables de entorno requeridas
    DB_NAME=
    DB_USER=
    DB_PASSWORD=
    DB_HOST=
    DB_DIALECT=
    DB_PORT=
    JWT_SECRET=
    PORT=
